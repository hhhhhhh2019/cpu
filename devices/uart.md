# UART

TYPE = 4

# Регистры

## Таблица регистров

размер - 3 байта

|адрес|название|описание|
|-:|:-:|-|
|0|STATE||
|1|DATA_IN||
|2|DATA_OUT||

## Описание регистров

### STATE

|бит|название|описание|
|-:|:-:|-|
|0|IN_READY|включен, если есть данные на вход|
|1|READ|если 1, переключается на следующие входные данные|
|2|WRITE|если 1, отправляет данные из DATA_OUT|
